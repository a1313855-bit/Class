//----------------------------匯出(一張表一個檔案)----------------------------//

//////////////////////////////////////匯出//////////////////////////////////////

Administration -> Data Export -> Export to Dump Project Folder -> 選擇要匯出的專案 -> 
Include Create Schema(打勾) -> Start Export -> Overwrite

//////////////////////////////////////刪除//////////////////////////////////////

右鍵 -> Drop Schemas

//////////////////////////////////////匯入//////////////////////////////////////

Administration -> Data Import -> Import from Dump Project Folder -> 
Import from Dump Porject Filder -> 選擇要匯入的專案 -> Start Import

* 如果出現 Unknown database 'company2' 代表他找不到當初存放這些資料的資料夾
-> 需手動新增 '  ' 單引號內的


//----------------------------匯出(多張表一個檔案)----------------------------//

//////////////////////////////////////匯出//////////////////////////////////////

Administration -> Data Export -> Export to Self-Contained File -> 選擇要匯出的專案 -> 
Include Create Schema(打勾) -> Start Export -> Overwrite

//////////////////////////////////////匯入//////////////////////////////////////

File -> Open SQL Script -> 選擇要匯入的SQL檔 -> 按下閃電符號
