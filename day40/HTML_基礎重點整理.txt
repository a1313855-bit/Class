HTML 基礎重點整理

【HTML 是什麼】 HTML（HyperText Markup Language） 用途： -
建立網頁結構（Structure） - 告訴瀏覽器畫面有哪些內容 -
不負責外觀（CSS） - 不負責互動（JavaScript）

可以把它想成： HTML = 房子的骨架

------------------------------------------------------------------------

【HTML 基本架構】

<!DOCTYPE html>
<!DOCTYPE html> - 宣告 HTML5 文件。

-   HTML 文件的根元素。

lang - 指定網頁主要使用的語言。 - 例如：zh-TW（繁體中文／台灣）。

------------------------------------------------------------------------

【head】

用途： - 放置網頁設定。 - 常見內容： meta title CSS JavaScript

------------------------------------------------------------------------

【body】

用途： - 放置真正顯示在畫面的內容。 - 例如：
標題、文字、圖片、按鈕、表單。

------------------------------------------------------------------------

【meta】

用途： - 指定文字編碼。 - UTF-8 可正確顯示中、英、日等文字。

------------------------------------------------------------------------

【title】

用途： - 設定瀏覽器分頁標題。 - 不是畫面中的標題。

------------------------------------------------------------------------

【標題】

h1~h6
用途： - 表示文章標題階層。 - 不是控制字體大小。

------------------------------------------------------------------------

【p】

用途： - 表示一個段落。

------------------------------------------------------------------------

【div】

用途： - 區塊容器。 - 常用來分區、排版、包住多個元素。

------------------------------------------------------------------------

【form】

用途： - 收集使用者輸入資料。

action - 指定表單送出的目的地。

method - 指定 HTTP 方法。

GET - 資料附加在網址後方。 - 適合查詢、搜尋。

POST - 資料放在 HTTP Request Body。 - 適合登入、註冊、新增資料。 - POST
不代表加密，真正安全依靠 HTTPS。

------------------------------------------------------------------------

【label】

用途： - 輸入欄位的說明文字。 - for 要對應 input 的 id。

------------------------------------------------------------------------

【input】

用途： - 建立輸入元件。

常見 type： - text - password - email - number - submit

常見屬性：

id - 元素唯一識別名稱。

class - 元素分類名稱，可重複。

name - 表單送出時的參數名稱。

value - 真正的值。 - 可作為預設值或送出的資料。

placeholder - 提示文字。 - 不是真正的值。

------------------------------------------------------------------------

【button】

用途： - 建立按鈕。 - 可放文字或 HTML。

------------------------------------------------------------------------

【select】

用途： - 建立下拉式選單。

用途： - 下拉式選單中的一個選項。 - value 是送出的值。 -
標籤內文字是使用者看到的內容。

------------------------------------------------------------------------

【a】

用途： - 建立超連結。

href - 指定超連結目標。

------------------------------------------------------------------------

【strong】

用途： - 表示重要內容。 - 通常會以粗體顯示，但重點是具有 HTML 語意。

------------------------------------------------------------------------

【br】

用途： - 強制換行。

------------------------------------------------------------------------

【HTML 註解】

用途： - 提供程式說明。 - 不會顯示在畫面中。 - 不可放密碼或機密資訊。

------------------------------------------------------------------------

【style】

用途： - 在 HTML 中撰寫 CSS。 - 通常放在 head 中。

style=“” - 行內樣式（Inline Style）。 - 只影響目前元素。

------------------------------------------------------------------------

------------------------------------------------------------------------

【HTML 常見屬性】

lang action method for type id class name value placeholder href style

------------------------------------------------------------------------

【HTML 三大重點】

1.  HTML：負責網頁結構。
2.  CSS：負責外觀。
3.  JavaScript：負責互動。

------------------------------------------------------------------------

【工程師最重要的觀念】

請理解每個標籤的責任，而不是死背語法。

head：放網頁設定。 body：放畫面內容。 form：收集並送出資料。
label：說明欄位。 input：讓使用者輸入。 id：唯一識別元素。
class：元素分類。 name：表單送出時的參數名稱。 value：真正送出的值。
placeholder：輸入提示文字。
